# UE5 RPG Demo
![SlashGif1](https://github.com/user-attachments/assets/60a7358f-e5ae-4248-899a-4e4e29f68c50)

This project was developed as a vehicle to build and showcase my skills working within Unreal Engine 5 C++. My goal with this project was to touch as many different areas of the game development process as I could, and I feel I succeeded in that regard. Going from a completely blank UE5 project to having a playable vertical slice of a game that uses photoreal assets was a humbling, informative, and truly inspiring experience. I am excited to take all of the experience gathered from this project, and apply it to my first publicly available indie release!
